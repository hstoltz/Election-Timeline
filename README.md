# U.S. Presidential Election Timeline

An interactive visualization of the presidential election process, from Election Day through Inauguration. Built to understand the constitutional framework, the 2020 election crisis, and post-reform safeguards.

## Features

- **Scaled Timeline**: Events are positioned proportionally to their actual dates
- **Year Toggle**: Switch between 2020 and 2028 to see pre/post-reform differences
- **Crisis Events Overlay**: Toggle to reveal the key events of the 2020 election crisis
- **Phase Filtering**: Focus on specific phases (certification, Electoral College, etc.)
- **Zoom Controls**: Adjust timeline scale for detailed viewing
- **Event Details**: Click any event for detailed information

## Key Events Covered

### Official Process
- Election Day
- Safe Harbor Deadline
- Electoral College Vote
- Congressional Joint Session (January 6)
- Inauguration Day

### 2020 Crisis Events
- Early attacks on mail-in voting (March-July 2020)
- Refusal to commit to peaceful transfer
- "Frankly, we did win" speech (Nov 4, 2:30 AM)
- GSA blocking transition
- Fake elector scheme (Dec 14)
- Trump-Raffensperger call ("find 11,780 votes")
- January 6 Capitol attack
- And more...

### Electoral Count Reform Act of 2022 Changes
- VP role clarified as "solely ministerial"
- Objection threshold raised to 1/5 of each chamber
- Governor certification requirement

## Deployment

This is a single HTML file with no dependencies—just deploy to GitHub Pages:

1. Push this repo to GitHub
2. Go to Settings → Pages
3. Select "Deploy from a branch"
4. Choose `main` branch, root folder
5. Your site will be live at `https://[username].github.io/[repo-name]/`

## Local Development

Just open `index.html` in a browser. No build step required.
